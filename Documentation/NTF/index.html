PoissonNTF - fluorescence microscopy data combining multiple excitations

PoissonNTF is a ImageJ plugin, similar to PoissonNMF, that allows to decompose lambda stacks of fluorescently labeled samples into the contribution of the individual dyes. In contrast to PoissonNMF, the NTF version allows to combine spectrally resolved data acquired using different excitation wavelength. The method is described in Neher et al., Biophysical Journal.
